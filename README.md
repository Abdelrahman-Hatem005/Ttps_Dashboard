# 📊 Restaurant Tips Data Dashboard

Interactive web application built with **Streamlit**, **Plotly Express**, and **Pandas** to perform Exploratory Data Analysis (EDA) on restaurant tipping behavior.

## 🚀 Live Demo
You can access the live interactive dashboard here:
👉 **[Click Here to Open the Dashboard](https://restaurant-tips-analysis.streamlit.app/)**

---

## 🧐 Features
- **Interactive Data Filtering:** Dynamic sliders and filters to explore total bills, tips, and days of the week.
- **Advanced Visualizations:** Beautiful, interactive scatter plots, bar charts, and pie charts generated via Plotly.
- **Key Metrics:** Instantly view averages and correlations regarding how different factors (like gender, smoking habits, or group size) impact tipping percentages.

## 🛠️ Tech Stack & Libraries
- **Python** (Core Logic)
- **Streamlit** (Web Interface & Deployment)
- **Pandas** (Data Manipulation & Analytics)
- **Plotly Express** (Interactive Charts)
